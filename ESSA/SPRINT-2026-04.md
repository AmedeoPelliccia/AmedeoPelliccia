## 1. MUSIC-MCC — Musical Cryptographic Communication

### 1.1 The Atomic Symbol

```
μ = (N, b, p)     v1: MonoBit
μ = (N, κ, p)     v2: MonKBit
μ = (H, κ, p)     v3: Harmonic object (note, chord, scale)
```

Three layers per unit: pitch identity (what it IS), activation κ (the DATA), stereo position p (spatial expression). The fundamental invariant:

```
𝒫 ∩ ℰ = ∅     payload and expression are disjoint
```

### 1.2 MonKBit = NeuronBit | KB=100

**KB = Knowledge Base** — the cardinality of the activation alphabet. Not kilobit, not kernel block. Algebraic base.

| Instance | KB | Bits/unit | Name |
|----------|-----|-----------|------|
| MonoBit | 2 | 1.000 | Binary bit on a note |
| MonKBit | 100 | 6.644 | Graded activation neuron |

Why KB=100: MIDI velocity (1–100), centesimal (2 decimal digits), percentage semantics (κ/100), TT alignment (100 steps = 1 Teknia Token).

The **monkbit** is a legitimate logarithmic unit: base 100. 1 monkbit = 2 dits exactly = 6.644 bits = 4.605 nats.

### 1.3 Shannon Anchor

Shannon's Theorem 2 (1948): H = −K Σ pᵢ log pᵢ is the UNIQUE measure of information. The monkbit inherits this uniqueness.

MCC instantiates Shannon's 5-part communication system:

- **Information source** → plaintext
- **Transmitter** → MCC encoder
- **Channel** → MIDI / audio / multi-sensory stream
- **Receiver** → MCC decoder
- **Noise** → cover music constraints + physical channel noise

Channel capacity at 120 BPM, MonKBit only: **13.3 bits/second** (predicted = observed).

### 1.4 Harmonic Extension (SPEC-007)

The MUSIC Unit generalizes to harmonic objects:

- Single note: N₍κ,p₎
- Chord: {C,E,G}₍κ,p₎ (multiplies voice count → multiplies capacity)
- Scale context: C-maj₍κ,p₎ (constrains allowed notes → lowers H_cover)

### 1.5 Five Compositional Styles

| Style | Progression | Character |
|-------|------------|-----------|
| ballad | I-vi-IV-V | Legato, breathing |
| jazz | ii-V-I turnaround | Swing, complexity |
| drammatico | i-iv-V-bVI | Harmonic minor tension |

### 1.6 Encoder v1 → v2 Pivot

v1 was correct but dead — "no trasmette nulla di emozionante."

v2 response: velocity is the ONLY dimension consumed by payload. Everything else — melody (voice-leading), harmony (real progressions), rhythm (variable patterns), bass (functional), stereo (separated), pedal (sustain) — is free for expression. Three MIDI tracks: melody (data-bearing), harmony (free), bass (free).

**12/12 round-trip tests pass** including empty string, UTF-8, 97-char sentence.

| Epitasis (II) | Payload: max variation | Crisis, climax |
