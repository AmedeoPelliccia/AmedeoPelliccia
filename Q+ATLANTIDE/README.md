## 1. Glossary of Terms

| Term | Definition | Governance Rule |
|---|---|---|
| Architecture Band [↗][pel] | Controlled numeric range used to classify a technology family. [↗][pel] | Defines **what** the technology is. [↗][pel] |
| Architecture Code [↗][pel-000] | Short identifier assigned to a band, such as ATLAS, STA, DTTA or CYB. [↗][pel-000] | Must remain unique inside the `000-999` register. [↗][pel-000] |
| ATLAS-1000 [↗][pel-000] | Umbrella name for the full Q-ATLANTIDE architecture register. [↗][pel-000] | Does not create a `1000` numeric band; operational codes remain `000-999`. [↗][pel-000] |
| Baseline [↗][pel] | Frozen version of the taxonomy accepted for reference, comparison and traceability. [↗][pel] | Changes require version increment and change record. [↗][pel] |
| Controlled Range [↗][pel] | Numeric interval reserved for a specific architecture family. [↗][pel] | Ranges must not overlap. [↗][pel] |
| Evidence Package [↗][pel] | Set of artefacts used to justify a node classification, maturity claim or gate decision. [↗][pel] | Required for programme-grade usage. [↗][pel] |
| Lifecycle Gate [↗][pel] | Acceptance point in the lifecycle where evidence is reviewed. [↗][pel] | Defines **when** a node becomes accepted. [↗][pel] |
| Node [↗][pel] | Individual architecture entry or subrange, for example `ATLAS 050-059`. [↗][pel] | Must carry architecture, authority and support mapping. [↗][pel] |
| ORB-Function [↗][pel] | Enterprise support function such as finance, legal, PMO or sustainability. [↗][pel] | Defines enterprise support, not technical ownership. [↗][pel] |
| Q-Division [↗][pel] | Technical centre of excellence responsible for engineering authority. [↗][pel] | Defines **who** owns technical authority. [↗][pel] |
| Trace Record [↗][pel] | Machine-readable or human-readable record linking node, evidence, owner, standards and gate. [↗][pel] | Defines **why** the node is valid. [↗][pel] |
| No-AAA Rule [↗][pel] | Prohibition of “AAA” as architecture, division, domain or interface layer. [↗][pel] | Use “Programme / Q-Division Interface” instead. [↗][pel] |
