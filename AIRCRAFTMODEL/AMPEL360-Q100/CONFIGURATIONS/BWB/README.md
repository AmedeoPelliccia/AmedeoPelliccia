# BWB — Blended Wing Body Configuration

> Sibling configuration of AMPEL360-Q100 under the shared OPT-IN framework.

## Geometry

Non-cylindrical centerbody with blended outer wing — a lifting-body design.

| Area | BWB Characteristic |
