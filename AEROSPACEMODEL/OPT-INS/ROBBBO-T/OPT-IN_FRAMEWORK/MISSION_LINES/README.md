# MISSION_LINES — ROBBBO-T product variants

Four variants share the OPT-INS framework instance:

| Line | Scope |
|---|---|
| COMMS | Communication satellites & relay |
| SAT | General-purpose satellite buses (EO/science/nav) |
| REPAIR | On-orbit servicing / refueling / assembly (OSAM) |
| DEBRIS | Active debris removal & end-of-life disposal |

Each line carries its own `LC01_PROBLEM_STATEMENT/` for line-specific KNOTs that are children of the product-level KNOTs.
