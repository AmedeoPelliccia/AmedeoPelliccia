# LC01 — Problem Statement (Q10 product-level)

Uncertainty orchestration for Q10 at the product level. Chapter-level LC01 nodes exist under each chapter's `SSOT/LC01_PROBLEM_STATEMENT/` and hold subject-specific KNOTs.

## Files

- `KNOTS.csv` — 7 seed KNOTs (MISS, LAUNCH, TPS, ECLSS, ABORT, CERT, PAX)
- `KNU_PLAN.csv` — expected knowledge units with target locations under the chapter tree
- `TIMELINE.csv` — MCR → SRR → PDR → TRR → FRR
- `RACI.csv` — per-KNOT responsibilities
- `TOKENOMICS_TT.yaml` — 2,100 TT pool total across 7 KNOTs
- `AWARDS_TT.csv` — distribution ledger (populated at closure)
